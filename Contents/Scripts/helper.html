A Helper Script

A Helper Script named helper.sh contents several bash or sh functions to help another scripts to display the followings:

  • line : Prints a line with specific characters
  • header: Prints header along with specific character line

Examples

$ source helper.sh 

$ line 
================================================================================

$ header Config
==================================== Config ====================================

$ line "#"
################################################################################

$ header Config "#"
#################################### Config ####################################
Download Script
curl "https://asispatra.github.io/Contents/Scripts/helper.sh" -o "helper.sh"
