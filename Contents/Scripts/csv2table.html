Print csv in tabular form: csv2table

There is a command named column which already helps you to print csv in tabular form. Here, we have used the same column command in csv2table.sh script which automatically take care of the header and prints in tabular form with border.

Examples

$ column -V
column from util-linux 2.32.1

$ cat csv.data
Instance,JTHREADS,JDURATION,Throughputs,Avg. Resp. Time,Min. Resp. Time,Max. Resp. Time,Error
1,20,1800,1984.6,9,0,131,0 (0.00%)
2,20,1800,1933.6,9,0,125,0 (0.00%)
3,20,1800,1976.7,9,0,159,0 (0.00%)
4,20,1800,1897.2,10,0,130,0 (0.00%)

$ bash csv2table.sh csv.data
---------+----------+-----------+-------------+-----------------+-----------------+-----------------+-----------------
Instance | JTHREADS | JDURATION | Throughputs | Avg. Resp. Time | Min. Resp. Time | Max. Resp. Time |            Error
---------+----------+-----------+-------------+-----------------+-----------------+-----------------+-----------------
       1 |       20 |      1800 |      1984.6 |               9 |               0 |             131 |        0 (0.00%)
       2 |       20 |      1800 |      1933.6 |               9 |               0 |             125 |        0 (0.00%)
       3 |       20 |      1800 |      1976.7 |               9 |               0 |             159 |        0 (0.00%)
       4 |       20 |      1800 |      1897.2 |              10 |               0 |             130 |        0 (0.00%)
---------+----------+-----------+-------------+-----------------+-----------------+-----------------+-----------------

Note: Please make sure you are using latest column version.
