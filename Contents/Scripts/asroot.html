Run commands as root user

This is a small script named asroot to run commands as root user.

Examples

./asroot COMMAND
./asroot "COMMAND"
./asroot "COMMAND ; COMMAND"

./asroot cp SRC DEST
./asroot "cp SRC DEST"

Note: Please make sure that root password is saved in PASS file in the same directory as asroot script.

echo ROOT_PASSWORD > PASS
Download Script
curl "https://asispatra.github.io/Contents/Scripts/asroot" -o "asroot"
