Wait Time Counter

sleep command helps us to put delay between two statements in shell script. Sometimes we want to find how much remaining time is there from the sleep time while the time is longer.

Here is a script named wait-time-counter.sh which helps in this kind of scenario.

#!/bin/bash

SLEEP_TIME=$(cat SLEEP_TIME 2>/dev/null || echo 10)
echo "### SLEEP_TIME: ${SLEEP_TIME} Second(s)"
echo "### Remaining Time: ${SLEEP_TIME} Second(s)"
i=0
while [ $i -lt ${SLEEP_TIME} ] ; do
  sleep 1
  i=$(expr $i + 1)
  echo -e $'\e[1F\e[2K### Remaining Time: '$(expr ${SLEEP_TIME} - $i)' Second(s)'
done

Note: You can put SLEEP_TIME in file or can adjust in the script.
