sed: Replace multi-line string

If there is \n in the pattern, sed does not match the pattern. As a solution we can replace \n with \r or \f before we use sed and later we can revert it back.

cat FILENAME | tr '\n' '\f' | sed 's/PATTERN1/PATTERN2/g' | tr '\f' '\n'
cat FILENAME | tr '\n' '\r' | sed 's/PATTERN1/PATTERN2/g' | tr '\r' '\n'

# Where PATTERN1 and PATTERN2 can contain '\n'
