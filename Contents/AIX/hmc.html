HMC Useful Commands

Release Partition Resources

chhwres -r proc -m <<SYSTEM_NAME>> -o r --id <<Partition_ID>> --procs <<Number_Of_Processors>>

Close HMC Console

rmvterm -m <<SYSTEM_NAME>> -p <<Partition_Name>>

Shows affinity score of managed system

lsmemopt -m <<SYSTEM_NAME>> -o currscore

Starts optimization for all LPARs on the entire system

optmem -m <<SYSTEM_NAME>> -t affinity -o start

Reference: http://aix4admins.blogspot.com/2015/01/dpo-magaed-systems-can-become.html

Shows progress status as percentage completed (progress=2 means 2%)

lsmemopt -m <<SYSTEM_NAME>>
